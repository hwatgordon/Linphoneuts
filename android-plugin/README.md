# Android Plugin

This directory contains the native Android plugin implementation.

## Structure

```
android-plugin/
├── plugin/            # Plugin source code
├── example/           # Example Android app
├── build.gradle       # Build configuration
├── settings.gradle    # Project settings
├── gradle.properties  # Gradle properties
└── README.md          # Android-specific documentation
```

## Development

1. Open the project in Android Studio
2. Build and run the example project
3. Implement plugin functionality

## API Documentation

See [API Reference](../docs/api/android.md) for detailed API documentation.

## Build Requirements

- Android API Level 21+
- Android Studio Arctic Fox+
- Kotlin 1.5+ or Java 8+
